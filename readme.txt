功能介绍：

1.	视频播放器

2.	应用: 主界面(home), 音乐(music), 视频(video), 相片(photo), 收音(fm), 
		录音(record), 电子书(ebook), 文件管理(explorer), 设置(setting);

3.	按键说明:

	开/关	:	拔动开关;
	0-9  	:	数字点歌/视频;
	播/停	:	播放/暂停, 确认;
	音量+/-	:	音量加减(文件管理时复用为上下键)
	上/下曲	:	上/下一曲
	movie	:	视频快捷键;
	music	:	音乐快捷键;
	FM	:	收音快捷键;
	Record	:	录音快捷键; 
	Menu	:	菜单键;
	Return	:	返回键;

4.	Line In功能;

	检测到Line In线插入自动启动 LineIn应用, 5s后自动关屏省电(忽视系统当前背光状态), 退出后自动亮屏;

	Line In插入时不能切换到其他应用;

5.	插卡/U盘自动播放;

6.	混响功能;
	
	当检测到外置麦克风插入时, 自动打开混响功能, 并通过I/O口控制开关管切断内置Mic, 防止啸叫;

7.	


注意事项:

1.	功放底噪音优化;

	(1)	屏驱pwm频率要在30kHz以上;

	(2)	功放IC模式切到D类, 只有FM时切为AB类;	

2.	FM调试:

	(1)	进入背光自动设置为5s, 自动搜台时自动设为1s, 搜台完成后, 自动亮背光;
		
		关背光是防止屏干扰到搜台和播放电台时的效果;
	
	(2)	进入FM后, 需要把功放IC的模式切换为AB类;

		切为AB类后, FM搜台效果大增, 播放电台清晰;
